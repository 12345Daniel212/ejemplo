METODOS DE INTERVALO

lso metodos de intervalo son tecnicas que se utilizan para encontrar soluciones
aproximadas de ecuaciones y sistemas de ecuaciones, limitando la solucion a un intervalo dado
en lugar de proporcionar un valor unico.

tambien son uitilez cuando se trabaja con funciones no lineales o complicadas, donde no es posible
resolver la ecuacion de manera analitica.

estos metodos solo proporcionan soluciones aproximadas y pueden requierir un numero variable de iteraciones
para converger hacia una solucion aceptable.

los metodos de intervalo tambien son conocidas como metodos CERRADOS, son tecnicas numericas utilizadas para localizar las raices
de una ecuacion aprovechando el hecho de que una funcion cambia de signo en la vecindad de una raiz real.

estos metodos se caracterizan por requerir 2 valores que deben encerrar o estar a ambos lados de la raiz.

METODOS DE GAUSS SEIDEL
este metodo se utiliza para resolver sistemas de ecuaciones lineales. se comienza con una estimacion inicial para todas las incognitas y
luego se realiza una serei de iteraciones donde en cada paso se resuelve una ecuacion para una de las incognitas, utilizando las estimaciones mas recientes
de las otras incognitas.

1. se despeja cada ecuacion para una incognita
2. propone una aproximacion para una incognita
3. se actualizan los valores uno por uno
3. se repite hasta que el error sea pequeno
